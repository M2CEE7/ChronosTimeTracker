## Introduction
Chronos Time Tracker is a comprehensive application designed to help users manage their daily activities efficiently. With features like timesheet entries, Pomodoro timer, and progress tracking, users can maintain a detailed log of their work hours, monitor their progress, and ensure they stay productive throughout the day.

## Features
- Login and Registration: Secure user authentication.
- Timesheet Entry: Log work hours with ease.
- Edit and View Entries: Modify and review timesheet entries.
- Pomodoro Timer: Time work sessions effectively.
- Daily Goals: Set and track minimum and maximum daily hours.
- Reports: Visualize progress with line graphs.
- Progress Bar: Track daily progress against set goals.
- Enhanced UI: Clean and modern design with a new app icon.
- Bottom Navigation Menu: Easily navigate to different sections of the app.

## Installation
To run the Chronos Time Tracker application, follow these steps:

2. **Open in Android Studio**:
   - Open Android Studio.
   - Select `Open an existing Android Studio project`.
   - Navigate to the cloned repository and select it.

3. **Build the Project**:
   - Ensure all dependencies are downloaded.
   - Sync the project with Gradle files.

4. **Run the Application**:
   - Connect an Android device via USB or use an Android emulator.
   - Click on the `Run` button or press `Shift + F10`.

## User Guide

### Login
1. **Open the App**: Launch the Chronos Time Tracker app on your device.
2. **Enter Credentials**: Provide your username and password.
3. **Login**: Click the Login button to access the main dashboard.

### Register
1. **Open Registration**: From the login screen, select the Register option.
2. **Fill in Details**: Provide the necessary information (username, email, password).
3. **Submit**: Click the Register button to create your account.

### Timesheet Entry
1. **Navigate to Timesheet Entry**: Use the bottom navigation menu and select the Add Timesheet Entry option.
2. **Add Entry**: Fill in the details for your work session.
3. **Save**: Click the Save button to log the entry.

### Editing Timesheet Entry
1. **View Entries**: Navigate to the List of Entries page.
2. **Select Entry**: Click on the entry you wish to edit.
3. **Edit Details**: Modify the necessary information.
4. **Save Changes**: Click Save to update the entry.

### Viewing Timesheet Entries
1. **Navigate to List of Entries**: Use the bottom navigation menu and select the List of Entries option.
2. **View Details**: Browse through the logged entries.
3. **Progress Tracking**: View the progress bar tracking your daily goals.
4, **Filter your List of Entries**: Filter you list of entries by clicking at the very top of the home page by categories or date range.

### Pomodoro Timer
1. **Open Pomodoro Timer**: Select the Pomodoro option from the bottom navigation menu.
2. **Set Timer**: Configure the duration for work and break periods.
3. **Start Timer**: Click Start to begin the session.
4. **Manage Breaks**: Use the break button to start/end break sessions as needed.

### Reports
1. **Navigate to Reports**: Select the Reports option from the bottom navigation menu.
2. **View Graphs**: Analyze your daily progress with a graph representing your logged hours.
3. **Summary**: Get a summary of your work patterns and productivity results by category or project with all time, last 7 days, last 30 days and custom range..
